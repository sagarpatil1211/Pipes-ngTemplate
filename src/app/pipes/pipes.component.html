Pipes

Pipes are referred as filters. It helps to transform data and manage data within interpolation, denoted by {{ symbol}}.

It accepts data, arrays, integers and strings as inputs which are separated by ‘ | ’ symbol.

1. date pipe

without pipe : {{today}}

toady's date : {{today | date}} = today | date

formatted date : {{today | date:'dd-MM-yyyy'}} = today | date:'dd-MM-yyyy'

hour date : {{today | date:'hh:mm:ss'}} = today | date:'hh:mm:ss'

short date : {{today | date:'shortDate'}} = today | date:'shortDate'

full date : {{today | date:'fullDate'}} = today | date:'fullDate'

2. Curreny pipe

Without pipe {{1000}}

USD {{1000 | currency}} = 1000 | currency

INR Rs {{1000 | currency:'INR'}} = 1000 | currency:'INR'

3. slice pipe

fruits array : {{Fruits}}

Start index : {{ Fruits | slice:2}} = Fruits | slice:2

Start and end index:- {{Fruits | slice: 1:4}} = Fruits | slice: 1:4

Negative index:- {{Fruits | slice: -2}} = Fruits | slice:-2

Negative start and end index:- {{Fruits | slice: -4:-2}} = Fruits | slice: -4:-2

4. number (decimal) pipe

Without pipe {{8.76543}}

number :- {{ 8.76543 | number}}

formatting number :- {{8.76543 | number:'2.1'}} = 8.76543 | number:'2.1'

formatting number :- {{8.76543 | number:'2.1'}} = 8.76543 | number:'2.1-4'

5. percent pipe

without pipe 1:- {{0.8234}}

percent :- {{0.8234 | percent}} = 0.8234 | percent

percent :- {{0.8234 | percent:'2.2'}} = 0.8234 | percent:'2.2'

6. json pipe

without pipe :- {{ fullName}}

json pipe :- {{fullName | json}}

{{ fullName | json }}

Chained pipes

We can combine multiple pipes together. This will be useful when a scenario associates with more than one pipe that has to be applied for data transformation.

Date, Uppercase and Lowercase are pre-defined pipes.

Date with uppercase :- {{today | date:'fullDate' | uppercase}}

{{ today | date:'fullDate' | uppercase }}

Date with lowercase :- {{today | date:'medium' | lowercase}}

{{ today | date:'medium' | lowercase }}

Custom pipe

As we have seen already, there is a number of pre-defined Pipes available in Angular but sometimes, we may want to transform values in custom formats.

We can create custom pipe using ng g p pipeName

When custom pipe generated , created class implements PipeTransform interface & must override transform method

square custom pipe :- {{5 | square}}

= {{ 5 | square }}
